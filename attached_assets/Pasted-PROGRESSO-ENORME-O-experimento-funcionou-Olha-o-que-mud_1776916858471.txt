sudo apt install -y python3-capstone

python3 tools/mips_inspect.py --gap 0x10047c

Esse comando vai:

    Detectar automaticamente que 0x10047c está no gap entre 0x100408 e 0x1004b8 (176 bytes)
    Calcular quantos % são zeros (= padding) vs código real
    Diagnosticar: se ≥90% zeros = "seguro pular", se ≤10% zeros = "código real, não pode pular"
    Mostrar o disassembly MIPS completo do gap

Bonus que vai te economizar muito tempo:

python3 tools/mips_inspect.py --list-gaps

Lista os 30 maiores buracos do recompilador no projeto inteiro. Vamos ver de uma vez quantos são, qual o tamanho típico, quantos são padding vs código real. Isso decide se a Ferramenta #4 deve ser um "gap patcher" automático ou não.
